# 大语言模型技术栈 - GPT部分

一本系统性介绍大语言模型LLM技术的书籍草稿，将定期(每月)发布更新版本。

## 📚 概览

本书包括三个主要部分：

- **GPT网络结构**：从神经网络计算图和数据流的视角出发，系统拆解GPT网络各分段的计算图结构与数据流向，完整揭示从原始文本输入到最终内容生成的端到端全过程与核心逻辑。
- **分层参考技术栈**：从分层参考模型与技术栈的视角出发，自底向上系统拆解大模型训练、推理等核心活动，针对大规模张量计算与序列建模等关键任务，构建清晰的结构化认知框架。
- **高级主题**：精选了一些发展快速、没能包括在上面两部分的内容，包括高性能推理技术、高级注意力和变换器架构技术等。
